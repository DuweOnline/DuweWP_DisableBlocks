Allows you to completely disable Gutenberg Blocks from the site.

== Description ==

Allows you to completely disable Gutenberg Blocks from your site with a handy list of all registered block - simply select the block you don't want and the plugin removes them from the editor for you. No more messing around with code plugins or functions files. Quick, simple and painless.

== Installation ==

Upload the plugin to your site, activate it, then go to Settings -> DuweWP Disable Blocks and select which blocks to disable.
